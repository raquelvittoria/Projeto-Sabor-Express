# 🍽️ Sabor Express

> Meu primeiro projeto em Python 🚀

---

## 📌 Sobre o projeto

O **Sabor Express** é uma aplicação simples em terminal desenvolvida com Python.

O objetivo do projeto é simular um sistema de gerenciamento de restaurantes, permitindo cadastrar e listar estabelecimentos.

Este projeto foi criado como parte do meu aprendizado inicial em programação.

---

## 🚀 Funcionalidades

- ✅ Cadastrar novos restaurantes  
- ✅ Listar restaurantes cadastrados  
- ⏳ Ativar restaurante *(em desenvolvimento)*  
- ❌ Sair do sistema  

---

## 🛠️ Tecnologias utilizadas

- Python 3  
- Biblioteca padrão (`os`)
